# Research Vault

Personal Obsidian knowledge base for aerospace structures & AI research, organized using the PARA method.

## Structure

```
01-inbox/        Unsorted notes — default landing zone
02-daily-notes/  Daily journal entries (YYYY/MM/)
03-projects/     Active research projects (time-bounded)
04-areas/        Ongoing domains & concepts (open-ended)
05-resources/    Reference materials — papers, tools, datasets
06-archives/     Completed projects
99-meta/         Templates, tag taxonomy, attachments
```

## Research Domains

- **Fatigue & Damage Mechanics** — CDM, multiscale fatigue life prediction, S-N curves
- **AI/ML for Structures** — PINN, Transformer, surrogate models, Bayesian inference
- **Digital Twin** — Virtual sensing, SHM, remaining useful life prediction
- **Composite Materials** — CFRP/GFRP, wind turbine blades, offshore structures
- **Virtual Certification** — EASA/FAA frameworks, probabilistic assessment

## Claude Code Integration

This vault is enhanced with [Claude Code](https://claude.ai/claude-code) skills:

| Skill | Description |
|-------|-------------|
| `/save` | Save web/YouTube/text content with auto-frontmatter and tag assignment |
| `/organize` | Classify and move notes to PARA folders with wikilink management |
| `/tag` | Manage tag taxonomy — add, merge, rename, audit |
| `/deep-dive` | Structured exploration of note concepts with Sequential Thinking |

All skills are integrated with the [Obsidian CLI](https://help.obsidian.md/cli) for direct vault interaction.

## Setup

1. Open this folder as an Obsidian vault
2. Install [Claude Code](https://claude.ai/claude-code)
3. Ensure Obsidian CLI is enabled (Settings → General → Command-line interface)
