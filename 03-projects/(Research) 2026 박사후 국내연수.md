# 2026년도 박사후 국내연수 사업

## 사업 개요

| 항목 | 내용 |
|------|------|
| **사업명** | 학문후속세대지원 - 박사후 국내연수 |
| **주무부처** | 교육부 (장관 최교진) |
| **전문기관** | 한국연구재단 (이사장 홍원화) |
| **사업목적** | 박사학위 취득 후 국내외 대학·연구소 등에서 연구를 수행할 수 있도록 연수 지원 |
| **연간 연구비** | 0.6억원 (간접비 포함) |
| **연구기간** | 1~3년 |
| **지원 규모** | 150개 과제 내외 / 4,500백만원 내외 |

## 지원 대상

- 국내·외 대학 **박사학위 취득 후 5년 이내**인 자
- 기준일: 2021.9.1. 이후 취득
- **대한민국 국적자**

## 신청 기간 (2차 접수)

| 구분 | 일정 |
|------|------|
| **연구책임자 신청 및 접수** | 2026.4.6.(월) 09:00 ~ 2026.4.13.(월) 14:00 |
| **주관연구기관 승인** | 2026.4.6.(월) 09:00 ~ 2026.4.15.(수) 14:00 |

## 신청 절차

1. **연구책임자 등록정보 갱신** (연구업적 등)
2. **연구계획서 온라인 접수** (연구책임자)
3. **주관연구기관 온라인 등록사항 확인·승인** (주관연구기관장)
4. **연구과제 신청 완료**

- 신청 사이트: [범부처통합연구지원시스템 (IRIS)](https://www.iris.go.kr)
- 세부사업별 신청요강을 반드시 확인하고 신청

## 평가 방법

- **온라인평가**
- 세부 평가항목 및 배점은 사업별 신청요강 확인

## 신청 자격 제한

- 교육부 및 과기정통부 소관 개인단위 기초연구사업은 연구책임자로 **1개 과제만 수행 가능**
- 교육부 및 과기정통부 소관 개인연구를 수행중인 연구책임자는 **신규 신청 불가**
  - 단, 수행 중인 과제가 신규과제 연구개시일로부터 10개월 이내에 최종 종료되는 경우 신청 가능
  - 박사후국내·외연수 사업의 경우 **별도 기준 적용**
- 동시 수행 가능 연구개발과제: 최대 5개 (연구책임자로서는 3개 이내)

## 내 상황 분석

### 기본 정보
- 연수기관: GIST (포닥)
- 연구주제: Multiscale Fatigue Analysis for Composites using AI
- 신청 기간: 1년
- 연구비: 6,000만원 (간접비 포함) → 간접비 제외 시 약 4,000만원 / 전액 인건비 확보도 가능

### 연구 키워드
- [[(Concept) Multiscale Fatigue Analysis|Multiscale Fatigue Analysis]] — 멀티스케일 피로 해석 (핵심 해석 방법론)
- [[(Concept) Digital Twin Composites|Digital Twin]] — AI 피로 해석 모델이 탑재되는 가상 복제본
- [[(Concept) Structural Health Monitoring|Structural Health Monitoring (SHM)]] — Digital Twin에 실시간 데이터를 공급하는 센서 기반 감지 체계
- [[(Concept) Virtual Certification|Virtual Certification]] — 시뮬레이션 기반 구조 인증, 최종 목표 (시험 비용 절감)
- [[(Research) 포닥 논문 계획|포닥 논문 계획]] — 4편 논문 구조 (Review + HyBC + PINN + Transformer)
- [[(Research) Paper-3 PINN Fatigue Degradation|Paper-3: PINN Degradation]] — 핵심 연구 산출물 (CDM 기반 PINN 피로 해석)
- [[(Research) Paper-4 Transformer Fatigue Analysis|Paper-4: Transformer Fatigue]] — 시계열 하중 이력 기반 RUL 예측

> **연결 구조**: AI 멀티스케일 피로 해석 → Digital Twin의 해석 엔진 → SHM 데이터로 모델 업데이트 → 시험 없이 수명 예측 = Virtual Certification

### 강점
- 산업계 연구 협력 경험이 풍부 → 디테일한 협력 계획으로 어필 가능
- 복합재료 + AI 융합 연구라는 차별화된 주제
- 연구비 확보 시 별도 물성시험 분석 수행 가능

### 약점
- 논문 실적 1편 — 실적 부족으로 탈락하는 경우가 많은 사업이라 불리
- 연구계획서의 완성도가 매우 중요

### 전략
- 학술적 목적의 연구주제가 선정에 유리 → 복합재료보다 넓은 범위의 AI 연구 결합
- 산업계 협력 성과를 기반으로 구체적인 협력 계획 제시
- 기존 수혜자(변재원, 류정현 등)에게 샘플 계획서 확보 및 조언 받기

### 주의사항
- 진행 중인 논문 작성이 밀리지 않도록 관리 (논문 4월 내 완료 목표)
- PI와 연구 방향 사전 상의 필수
